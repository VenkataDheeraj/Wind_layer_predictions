## Env Setup

If you wish to run this project independently on your local host for test purposes, you will need to set up seval things
first.

1. The required libraries listed in the `requirements.txt` file, you should set up a cond environment with the required
   versions of the libraries by first:

         conda create -n atmos-gen
  
   Then use the `atmos-gen` environment by

         conda activate atmos-gen

   Then install the various dependencies

         conda install -c conda-forge numpy~=2.0.1 hcipy~=0.6.0 matplotlib~=3.9.2 h5py~=3.11.0 tqdm~=4.66.5 scikit-image~=0.24.0 opencv~=4.10.0

2. You can then set your interpreter for your PyCharm IDE to the one in the conda environment by following the
   instructions at
   the [PyCharm Config Conda Page](https://www.jetbrains.com/help/pycharm/conda-support-creating-conda-virtual-environment.html)
